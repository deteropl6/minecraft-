ДОБАВЛЕННЫЕ ДАТАПАКИ:
1.advanced villager trades - меняет сделки жителей. (цены, добавляет новые сделки)
2.foodbuff - улучшает эффекты от еды.
3.leashvillager - жителей можно посадить на поводок.
4.mending neft - убирает починку из игры (предметы удаляются).
5.ores generation - корректирует генерацию руд в игре.
6.powder - бетон можно делать в котле с водой.
7.spawnanimation - добавляет спавн анимацию мобов.
8.totemfix - тотемы не выпадают, тотемы и золотые яблоки крафтятся.
9.trident - трезубец можно скрафтить.
10.weight armor - баф\дебафы скорости у брони.
11.bo - добавляет ботинки в сделке у бронника с супер способностями. 
12.wasd - пак необходимый для работы bo.
